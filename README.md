# cortical_geometry
Ref:Cross-Species Cortical Geometry Reveals Conserved Gradients Across Primates and Human-Specific Expansion
Yifan Wei, Yufan Wang, Tingwei Wei, Xiaoting Lu, Deying Li, Chet C. Sherwood, Yu Zhang, Chen Cheng, Tianzi Jiang, Lingzhong Fan, Luqi Cheng
bioRxiv 2025.11.28.691123; doi: https://doi.org/10.1101/2025.11.28.691123

- `Deformation`: Contains spherical deformation files for aligning brain data across species.
- `Expansion`: Includes cortical expansion maps for chimpanzee to human, macaque to human, and macaque to chimpanzee.
- `Geometric_gradients`: Holds geometric gradients for human, chimpanzee, macaque, and extended primate species.
- `MyelinMap`: Contains myelin maps for human, chimpanzee, macaque, and deformed myelin maps that have been aligned across species using spherical deformation.
- `SCSI`: Provides the Structural Connectivity Similarity Index (SCSI) for chimpanzee to human and macaque to human.
- `Surface`: Includes midthickness cortical surfaces at 10k and 32k resolution.
